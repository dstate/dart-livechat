{action: [ACTION], data: [DATA]}

[ACTION] - integer
[DATA] - dictionnary

--------------------------------------------------------------------------------

[ACTION]
+-------------------------------+
| Value | Action                |
+-------------------------------+
| 0     | Set nickname          |
| 1     | Send message          |
+-------------------------------+

--------------------------------------------------------------------------------

[DATA] with [ACTION] == 0

{nickname: [NICKNAME]}
[NICKNAME] - string

--------------------------------------------------------------------------------

[DATA] with [ACTION] == 1

{message: [MESSAGE]}
[MESSAGE] - string
